## Running the UI and Lambda Entry Endpoint Locally
1. `git clone`
2. `cd my-auth-playground`
3. Run Docker
4. `sam local start-api`
5. `cd oauth2-app && npm i`
6. `ng serve`
7. Navigate to [localhost:4200](http://localhost:4200)
